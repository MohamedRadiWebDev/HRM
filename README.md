# نظام الحضور والانصراف (واجهة محلية)

واجهة React محلية تعمل بدون أي خادم وتعيد إنتاج منطق حضور/انصراف وشيتات الرواتب وفق المتطلبات.

## التشغيل المحلي

```bash
npm install
npm run dev
```

## الاختبارات

```bash
npm test
```

## ملاحظات
- جميع البيانات تحفظ في الذاكرة فقط، التحديث أو إعادة التحميل يمسح كل شيء.
- يمكن استيراد قواعد الحالات الخاصة عبر ملف `special_rules.xlsx`.
- الأعمدة غير المطابقة للقالب يتم وسمها في شاشة التصدير بـ **Needs Mapping**.
